# Grafici a barre{#bar-charts}

Le visualizzazioni dei grafici a barre consentono di visualizzare i dati delle metriche e delle dimensioni con ciascun elemento dati rappresentato come una barra orizzontale sul grafico. I grafici a barre consentono la visualizzazione simultanea di fino a 200 elementi di dati univoci sul grafico. I valori delle dimensioni sono etichettati lungo l’asse y, mentre gli incrementi delle metriche sono etichettati lungo l’asse x. La barra per ogni valore di dimensione è etichettata con il valore della metrica esatta.
